# 🚗 RideMate Pro - Smart Carpool Frontend Website

RideMate Pro is a **startup-style smart carpool / ride sharing frontend website** built using **HTML, CSS, and JavaScript**.  
It is designed for **college students, office employees, and daily commuters** to find and offer rides in a modern, responsive, and user-friendly interface.

This project is a **real-world product-based frontend project** that can be used for:

- 🎓 College project submission
- 💼 Resume / portfolio showcase
- 🚀 Frontend development practice
- 📌 Placement project demo
- 🌍 Real-world problem solving

---

## ✨ Features

- ✅ Premium responsive landing page
- ✅ Smart carpool / ride sharing UI
- ✅ Find Ride form
- ✅ Offer Ride form
- ✅ Fare Calculator
- ✅ Available rides section
- ✅ Ride category filters
- ✅ Verified driver cards
- ✅ User dashboard preview
- ✅ Ride history table
- ✅ FAQ accordion
- ✅ Newsletter subscription
- ✅ Login / Signup modal
- ✅ Dark / Light mode toggle
- ✅ Toast notifications
- ✅ Back to top button
- ✅ Smooth reveal animations
- ✅ Glassmorphism modern UI
- ✅ Mobile responsive design
